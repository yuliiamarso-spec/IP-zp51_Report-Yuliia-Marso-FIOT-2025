Контрольні питання з CSS

  1. Для чого використовуються каскадні таблиці стилів?

    2. Каскадні таблиці стилів (CSS) використовуються для відокремлення презентаційного оформлення від HTML-коду, дозволяючи змінювати вигляд елементів без зміни структури сторінки.

  2. Які методи визначення стилів у HTML-документі ви знаєте?

    3. В HTML можна визначати стилі трьома способами:

    • Inline-стилі — через атрибут style у тегу: <p style="color:red;">Текст</p>
    • Внутрішні стилі — через блок <style>...</style> у <head>
    • Зовнішні стилі — через файл style.css підключений тегом <link rel="stylesheet" href="style.css">
  3. У чому різниця між каскадуванням і спадкуванням?

    4. Каскадування: якщо до елемента застосовується кілька CSS-правил, браузер обирає пріоритетне на основі специфічності, порядку та важливості !important

    Спадкування: властивості, як-от шрифт або колір тексту, автоматично передаються від батьківського елемента до дочірніх, якщо не перевизначені.

  4. Як оформлюються селектори тегу і класу?

    5. Селектор тегу задає стилі для всіх тегів певного типу:

    p {
    font-size: 16px;
    color: #444;
}

    Селектор класу застосовується до елементів із певним класом:

    .highlight {
    background-color: #ffe5e5;
    font-weight: bold;
}
  5. Для чого використовується ідентифікатор?

    6. Ідентифікатор id застосовується для унікального позначення елемента. Дає можливість задавати стилі для одного конкретного блоку та звертатися до нього через JavaScript.

    Приклад елемента з ідентифікатором:

    <div id="output">
    Тут відображається вибраний контент
</div>

    Приклад стилю:

    #output {
    padding: 20px;
    background-color: #f5f5f5;
    border-radius: 12px;
}
  6. Навести приклади оформлення текстових елементів.

    7. Приклади оформлення тексту:

    body {
    font-family: Arial, sans-serif;
}

.logo-text {
    font-size: 50px;
    font-weight: bold;
    font-family: 'Montserrat', sans-serif;
}

.slogan {
    font-family: 'Nunito', sans-serif;
    color: #b04f6f;
}

.bottom-bar .menu a {
    color: white;
    font-weight: bold;
    text-decoration: none;
}
