CSS: Шрифти, Текст, Таблиці, Фон, Контур, Списки, CSS просунутий

1. Шрифти та текст

body {
    font-family: Arial, sans-serif;
}

.logo-text {
    font-size: 50px;
    font-weight: bold;
    font-family: 'Montserrat', sans-serif;
}

.slogan, .register {
    font-family: 'Nunito', sans-serif;
}

Встановлюємо базовий шрифт для всього тіла, великі заголовки і тексти кнопок/слоганів — окремими шрифтами.

2. Таблиці

.donations-table {
    width: 100%;
    border-collapse: separate;
    border-spacing: 0;
    background-color: #fff;
    border-radius: 12px;
    box-shadow: 0 6px 15px rgba(0,0,0,0.15);
}

.donations-table th, .donations-table td {
    padding: 15px 20px;
    text-align: left;
}

Таблиці мають скруглені кути, тінь, відступи в клітинках.

3. Фон

.top-bar {
    background-color: #000;
    color: white;
}

.middle-bar {
    background-color: #d18b97;
}

.photo-wrapper .background-photo {
    filter: brightness(65%);
}

Встановлюємо кольори смуг, затемнення фото для читабельності тексту поверх зображення.

4. Контур та тінь

.donate-container {
    background: rgba(255,255,255,0.25);
    backdrop-filter: blur(14px);
    box-shadow: 0 6px 18px rgba(0,0,0,0.28);
    border-radius: 22px;
}

Скляна картка донату: напівпрозорий фон, ефект blur і тінь.

6. CSS просунутий (псевдокласи, анімації)

.submit-btn:hover {
    background: #b65672;
    transform: translateY(-3px);
    box-shadow: 0 6px 12px rgba(0,0,0,0.22);
}

@keyframes fadeIn {
    from { opacity: 0; transform: translate(-50%, -46%); }
    to { opacity: 1; transform: translate(-50%, -50%); }
}

Анімація появи картки донату та ефекти при наведенні на кнопку.
