Селектори класу

Стилізація елементів за класом:

.page-wrapper {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}

.donate-list-left,
.donate-list-right {
    flex: 1;
    background: #ffffffdd;
    padding: 25px;
    border-radius: 16px;
}

Блок .page-wrapper використовує flex-контейнер, а списки .donate-list-left та .donate-list-right мають фон, паддінг і скруглення.

Ще приклад:

.submit-btn {
    width: 100%;
    padding: 14px 0;
    border: none;
    border-radius: 14px;
    background: #ca6c89;
    color: white;
}

Всі кнопки з класом .submit-btn будуть рожевими з білим текстом і скругленими кутами.
