Селектори ідентифікаторів

Стилізація елементів за id:

#output {
    padding: 20px;
    background-color: #ffffffdd;
    border-radius: 12px;
    box-shadow: 0 3px 10px rgba(0,0,0,0.1);
}

Елемент з id="output" отримає білий фон, скруглені кути і тінь.

Ще приклад з форми донату:

#donate-form input {
    width: 85%;
    padding: 15px;
    border-radius: 14px;
    background: #ffffffcc;
}

Поле вводу з id="donate-form" буде світлим і скругленим.
