Селектори тегу

Стилізація елементів за тегом:

header {
    width: 100%;
}

footer {
    width: 100%;
    font-family: Arial, sans-serif;
}

Всі елементи <header> будуть займати всю ширину, а <footer> матиме шрифт Arial.

Ще приклад для абзаців:

p {
    line-height: 1.5;
    color: #444;
}

Всі <p> матимуть відстань між рядками 1.5 і текст темного кольору.
