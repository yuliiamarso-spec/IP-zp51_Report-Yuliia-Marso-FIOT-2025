Контрольні питання

  1. Яка структура HTML-документу
    2. HTML-документ складається з наступних основних частин: <!DOCTYPE html> <html></html> <head></head> <body></body>
    Head містить мета-інформацію, підключення CSS та скриптів. Body містить видимий контент сторінки, такий як тексти, зображення, таблиці та форми.
  2. Дати визначення тега
    3. Тег — базовий елемент HTML, який позначає структурні або семантичні частини документа. Приклади: <p></p> <h1></h1> <img></img>
  3. Які теги використовуються при роботі з таблицями
    4. Основні теги таблиць: <table></table> <caption></caption> <thead></thead> <tbody></tbody> <tfoot></tfoot> <tr></tr> <th></th> <td></td>
  4. Які теги використовуються при роботі з зображеннями
    5. Тег зображення: <img></img> Атрибути: src alt width height
  5. Які теги використовуються при роботі зі списками
    6. Для списків: <ul></ul> <ol></ol> <li></li>
  6. Перерахувати засоби опису таблиць в HTML
    7. Теги таблиць: <table></table> <caption></caption> <thead></thead> <tbody></tbody> <tfoot></tfoot> <tr></tr> <th></th> <td></td>
    Атрибути комірок: colspan rowspan align valign
    Стилізація таблиць через CSS для рамок, кольорів, відступів та шрифтів
